# FD Support AI

FD Support AI is a multi-tenant AI-powered customer support infrastructure built by Freener’s Digital Services.

## Features
- Multi-tenant architecture
- AI-powered knowledge replies
- Agent dashboard system
- Chat limit engine (1,000 chats/month default)
- Escalation to human agents
- Priority tagging
- Resolution history tracking

## Tech Stack
- Node.js
- Express
- MongoDB
- OpenAI API
- Railway Deployment

## Vision
To provide scalable AI-driven customer support infrastructure for modern businesses.
